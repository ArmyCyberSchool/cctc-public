Local Port Forwarding with IPv6

- The major difference with IPv6 ssh tunneling is that the colon in the IPv6 address must be understood as part of the address field (not th colon to specify the port info) this is done by enclosing the address in square brackets which must then be escaped with backslashes:

ssh -L 9001:\2001:DEAD:BEEF::1\}80 user@2001:1234:1234::1
(For link-local address(fe80::0 range) specify the interface following the SSH GWY)
