Exfiltration

(obj. 19.1 - 19.3)

Exfiltration done via outbound connections are very common, and hard to counter

at the pereimeter since this is an area that has been created to

focus on incoming threats

Common protocols are used to mask data exfiltration:

HTTP/HTTPS, FTP and DNS for example easily blend into normal

outbound network traffic, thus obfuscating the attacker's actions

Credentials

obj. 19.3: Explain the benefits of acquiring domain credentials)

Common means for stealing credentials: Social Engineering, Phishing, Reusing

stolen passwords/shared credentials, brute force, securityquestion reuse

Credentials can provide adversary with:

Remote Access - gain remote entry into networks using Virtula Private networks (VPN)

or remote access protocols, like RDP and VNC

Movement within a network - Stolen credentials (especially domain admin credentials)

can allow an attacker to move laterally within a network, or potentially

deeper into higher security areas

Cloud Access - Cloud services that can be accessed via domain credentials

provide a means for an attacker to easily access valuable data, especially

as organizations move more and more information into the cloud
